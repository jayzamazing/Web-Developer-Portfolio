Projects

Restaurant POS

This project is a mock restaurant pos system. It allows a logged in user to select a table, add various checks to a table, add orders to a specific check, delete empty checks, and cash out a check. This project uses Feathers, HTML5, CSS3, JavaScript, AngularJS, Node.js, Express, jQuery, Bootstrap, bcrypt, body-parser, Mongoose, MongoDB, Passport, Babel, Mocha, Chai, Less, UglifyJS, and webpack.

Music Quiz App

This is an web app that asks the user to identify the artist and album a set of lyrics are from. The user can also play the music for a clue to answer the question. This was done in HTML5, CSS3, Bootstrap, Handlebars, Run-Auto, Request, jQuery, JavaScript, Json, Node, Express, npm, Spotify Web API, Musixmatch API, and Spotify Web API Node.

Awesome Shopping List

This is a shopping list app that allows for the creation of various lists with their own items. The lists are stored locally, so coming back to the page the user will be able to see their previous list. In the shopping list section, the selected item is highlighted. The shopping list items section highlights any list item that is crossed off the list. Deleting items in both list removes it from storage. It uses HTML5, CSS3, jQuery, CSS3, and Web Storage API.
