Projects

Music Quiz App

This is an web app that asks the user to identify the artist and album a set of lyrics are from. The user can also play the music for a clue to answer the question. This was done in Html, CSS, Bootstrap, Handlebars, Run-Auto, Request, Jquery, Javascript, Json, Node, Express, Npm, Spotify Api, MusixMatch Api, and spotify-web-api-node.

Awesome Shopping List

This is a shopping list app that allows for the creation of various lists with their own items. The lists are stored locally, so coming back to the page the user will be able to see their previous list. In the shopping list section, the selected item is highlighted. The shopping list items section highlights any list item that is crossed off the list. Deleting items in both list removes it from storage. It uses Html, Css, Jquery, Javascript, and LocalStorage.

Party App

An Android app that shows restaurants and pubs in the near vicinity, and allows you to drive to the locations. Uses Android, Java, Google Maps, Google Places, and JSON.
